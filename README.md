# База данных для магазина гитар (Guitar Store Database)

Проект демонстрирует навыки проектирования реляционных баз данных и написания SQL-запросов на примере магазина гитар.

## Схема базы данных
guitar_store
├── users # Пользователи (покупатели)
├── categories # Категории товаров
├── products # Товары (гитары)
├── orders # Заказы
└── order_items # Состав заказов (связь many-to-many)
